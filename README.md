# URL Encoder / Decoder

This tool is a simple and easy online tool to encode or decode URLs.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm start
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
